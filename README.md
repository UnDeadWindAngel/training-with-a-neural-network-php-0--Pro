Проект написанный под руководством нейросети в учебных целях, от стажера по PHP до профессионала. В ходе данного проекта был взят простой форум за основу и проделан путь написания форума от простого PHP legacy до написания и разверстки своих классов маршрутизаторов, промежуточного ПО, с использованием стандартов текущего времени (PSR-4 как пример).
